# jawf

## 项目简介
`jawf` 是一个面向需求到工作流自动化原型系统。  
系统围绕“自然语言需求理解与流程编排执行”展开，支持从需求输入到工作流执行的完整主流程。

## 核心流程
用户提出需求 -> 意图识别 -> 追问并澄清需求 -> 生成 Workflow -> 修改并执行 Workflow

## 功能完成情况
作为课程设计，目前仅完成主体功能。
